Para rodar, é necessário xampp versão 3.2, que vem com o PHP 5.0.
Para facilitar, as versões do xampp estão disponíveis no link: https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/

Nesse projeto, foram usadas funções antigas do mysql no PHP, tais como mysql_connect, etc, por isso a necessidade de um PHP antigo, pois nos atuais essas funções foram removidas.

O banco e as tabelas estão conectadas no arquivo para teste, mas caso queira usar codigos como base, por favor conecte seu banco.

O site foi feito com:
PHP 5.0,
MYSQL,
JavaScript,
Bulma.

E possui versabilidade para aparelhos celulares.
